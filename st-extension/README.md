# ProxyTavern SillyTavern Extension (Phase B scaffold)

Install via SillyTavern extension installer using repository URL.

## Package layout
- `manifest.json` - extension metadata and JS entrypoint.
- `index.js` - minimal settings registration scaffold.
- `settings.json` - default values for endpoint and port.

## Validation
From repository root:

```bash
python scripts/validate_st_extension.py
```
